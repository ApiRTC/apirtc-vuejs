# apirtc-vuejs

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```


```
npm run build

git add docs/*

git status

git commit -a -m "deploy"

git push origin main
```
